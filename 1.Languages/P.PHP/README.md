## :b: Functional Programming

- HOF

```php
$ php -a
Interactive shell

php > function sum(int $number1, int $number2) {
        return $number1 + $number2;
      }
php > var_dump( sum(sum(3, 4),sum(5,5)));
int(17)
php > echo sum(sum(3, 4),sum(5,5));
17
php >
```

- Map

